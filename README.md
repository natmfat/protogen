# protogen

Generate TypeScript types and [Zod](https://github.com/colinhacks/zod) validators from protobuf schemas. There are of course, better options like [ts-proto](https://github.com/stephenh/ts-proto)

|Syntax|Implemented|
|-------|-----------|
|Message|✅|
|Reserved|✅|
|Comments|❌|
|Enums|❌|
|Import|❌|
|`repeated`|❌|
|`optional`|❌|
|Nested messages|❌|
|`google.protobuf.Any`|❌|
|`oneof`|❌|
|Maps|❌|
|`option`|❌|
|`extend`|❌|
